# Discorder v2.0.13 - Portable Windows Discord bağlantı yöneticisi

Tarih: 2026-06-12

## Kısa Özet

v2.0.13, sertifika olmadan da uygulama içi güncellemenin çalışması için güncelleme güven sınırını GitHub release doğrulaması, asset digest, SHA-256 dosyası ve manifest eşleşmesi üzerine taşır. Authenticode sertifikası varsa workflow imzalı paket üretmeye devam eder; sertifika yoksa imzasız paket yayınlanır ama hash/manifest doğrulaması zorunlu kalır.

## Neler Değişti?

- Otomatik update artık Authenticode sertifikası olmadan da hazırlanabilir.
- Paket yine yalnızca resmi `ucsahinn/discorder` GitHub release yolundan kabul edilir.
- GitHub asset digest ile `.sha256.txt` dosyası eşleşmek zorundadır.
- ZIP içindeki `discorder.update-manifest.json` sürüm ve dosya listesiyle doğrulanır.
- Manifest dışı dosya, path traversal, yanlış sürüm ve hash uyuşmazlığı reddedilir.
- Portable update mevcut çıkarılmış Discorder klasörünü hedeflemeye devam eder.

## Neden Önemli?

Kod imzalama sertifikası yokken de kullanıcı GitHub Releases üzerinden yeni sürümü alabilir ve uygulama içinden güncelleyebilir. Bu mod, imza güveni sağlamaz; güven sınırı GitHub release sahibi, HTTPS, GitHub asset digest, SHA-256 dosyası ve manifest doğrulamasıdır.

## Kurulum

1. GitHub Releases sayfasından `Discorder-2.0.13-win-x64.zip` arşivini indirin.
2. Yanındaki `Discorder-2.0.13-win-x64.sha256.txt` dosyasıyla arşivi kontrol edin.
3. ZIP içeriğini kalıcı bir klasöre çıkarın.
4. `Discorder.exe` dosyasını çalıştırın.

## Uygulama İçi Güncelleme

- **Güncelle** önce yeni sürüm olup olmadığını denetler.
- Yeni sürüm varsa **Yükle** düğmesi görünür.
- **Yükle**, denetlenen sürümün ZIP paketini indirir.
- Paket SHA-256, GitHub digest ve manifest kontrollerinden geçerse uygulanır.
- Başarılı olursa yeni `Discorder.exe` yeniden başlatılır.

## Güvenlik Notu

Bu sürüm public Authenticode sertifikası olmadan yayınlanabilir. Windows SmartScreen yine uyarı gösterebilir. Bu durum güncelleme paketinin SHA-256/manifest doğrulamasından geçtiği gerçeğini değiştirmez, fakat yayıncı kimliği güvenini sağlamaz.
