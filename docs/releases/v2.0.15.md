# Discorder v2.0.15 - Tanılama ve bağlantı kapsamı düzeltmeleri

Tarih: 2026-06-12

Not: Bu sürüm tarihsel kayıt olarak korunur. Güncelleme ilerleme penceresi, helper staging, ilk kurulum indirme görünürlüğü ve Cloudflare WARP profil hazırlığı düzeltmeleri `v2.0.18` sürüm hattında yayınlandı.

## Kısa Özet

v2.0.15, gerçek tanılama paketlerinden yakalanan iki alanı düzeltir: kötü DNS/internet koşullarında indirme davranışı daha dayanıklı hale gelir, Tarayıcı modu kapalıyken Discord web kapsamı daha sıkı korunur ve uygulama kapanışı daha hızlı tepki verir.

## Neler Değişti?

- Tarayıcı modu kapalıyken Discord web erişimini sınırlayan firewall kapsamı güçlendirildi.
- Chrome, Edge, Brave, Firefox, Opera, Vivaldi ve Chromium için kurulum yolu keşfi genişletildi.
- Browser path keşfi artık sabit kurulum yollarına ek olarak registry App Paths, PATH komutları ve çalışan browser süreçlerini de kontrol eder.
- Discord domain listesi PTB, Canary, status/support, updates ve ek Discord yüzeyleriyle genişletildi.
- WireSock CLI kapatılırken penceresiz süreçlerde gereksiz bekleme kaldırıldı; bağlantı kapatma ve uygulama çıkışı daha hızlı tepki verir.
- HTTP indirme ayarı yavaş bağlantılar için daha toleranslı hale getirildi.
- GitHub DNS hatası geçici olduğunda doğrulanmış indirici retry davranışı testle kilitlendi.

## Neden Önemli?

Kullanıcı Tarayıcı modunu açmadıysa beklenti nettir: Discorder sadece Discord uygulaması bağlantısını yönetir, Discord web tarafı kapsam dışında kalır. Bu sürüm, farklı browser kurulum yollarında ve çalışan browser süreçlerinde bu beklentiyi daha iyi uygular.

## Kurulum

1. GitHub Releases sayfasından `Discorder-2.0.15-win-x64.zip` arşivini indirin.
2. Yanındaki `Discorder-2.0.15-win-x64.sha256.txt` dosyasıyla arşivi kontrol edin.
3. ZIP içeriğini kalıcı bir klasöre çıkarın.
4. `Discorder.exe` dosyasını çalıştırın.

## Uygulama İçi Güncelleme

- **Güncelle** önce yeni sürüm olup olmadığını denetler.
- Yeni sürüm varsa **Yükle** düğmesi görünür.
- **Yükle**, denetlenen sürümün ZIP paketini indirir.
- Paket SHA-256, GitHub digest ve manifest kontrollerinden geçerse uygulanır.
- Başarılı olursa yeni `Discorder.exe` yeniden başlatılır.

## Güvenlik Notu

Bu sürüm public Authenticode sertifikası olmadan yayınlanabilir. Windows SmartScreen uyarısı görülebilir. Güncelleme paketi yine GitHub release yolu, GitHub asset digest, SHA-256 dosyası ve manifest doğrulaması olmadan uygulanmaz.

## Bilinen Sınır

İlk kurulumda GitHub'a hiç erişilemiyorsa `wgcf.exe` henüz indirilemediği için bağlantı kurulamaz. Dosya bir kez doğrulanıp cache'e girdikten sonra sonraki çalıştırmalarda aynı doğrulanmış dosya tekrar kullanılır.
