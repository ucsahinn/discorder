# Discorder Public Discovery Metinleri

Bu sayfa README, GitHub açıklaması, release notu ve kısa ürün tanıtımı için tekrar kullanılabilecek doğal metinleri toplar. Amaç arama motorlarına ve GitHub kullanıcılarına ürünün ne olduğunu net anlatmaktır; spam veya resmi ortaklık iddiası yoktur.

## Kısa Başlık Adayı

Discorder - Windows için portable Discord bağlantı yöneticisi

## Meta Açıklama Adayı

Discorder, Windows'ta Discord bağlantısını sade arayüzle yöneten portable masaüstü uygulamasıdır; güvenli güncelleme ve tanılama akışı sunar.

## Tek Cümlelik Özet

Discorder, Windows kullanıcılarının Discord bağlantısını sade, portable ve güvenli bir arayüzle yönetmesine yardımcı olan masaüstü uygulamasıdır.

## Özellik Mesajları

- Discord bağlantısını tek ekrandan başlatır, durdurur ve takip eder.
- Tarayıcı modu, arka planda çalışma, tanılama ve güncelleme akışını sadeleştirir.
- Portable ZIP olarak çalışır; kullanıcı çıkarılmış klasörü kontrol eder.

## Güven Mesajları

- Sistem DNS ayarını kalıcı değiştirmez.
- Uygulama içi otomatik güncelleme paketlerini GitHub release yolu, asset digest, SHA-256 ve manifest kontrolleriyle doğrular.
- Discord, Cloudflare veya WireSock tarafından üretilen resmi bir ürün olduğunu iddia etmez.

## Kurulum ve Güncelleme Mesajları

- GitHub Releases üzerinden en güncel ZIP arşivini indirin ve yayınlanan SHA-256 dosyasıyla kontrol edin.
- ZIP'i kalıcı bir klasöre çıkarıp `Discorder.exe` dosyasını çalıştırın.
- Güncelleme önce denetlenir; yeni sürüm varsa kullanıcı onayıyla yüklenir.

## Kapsanan Kullanıcı Soruları

README ve dokümanlar şu kullanıcı sorularını doğal biçimde yanıtlayacak şekilde yazılmıştır:

- Discorder nedir ve Windows'ta ne işe yarar?
- Discord bağlantısı neden ayrı bir araçla yönetilmek istenebilir?
- Portable ZIP nasıl indirilir, çalıştırılır ve güncellenir?
- Uygulama içi güncelleme neden önce denetleme sonra yükleme şeklinde çalışır?
- Discorder sistem DNS'ini, tüm cihaz trafiğini veya Discord hesabını değiştirir mi?
- Tanılama paketi hangi durumlarda hazırlanır?
